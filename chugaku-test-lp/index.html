全国統一中学生テスト 岡山校舎版

岡山から挑戦する、全国の舞台。

全国の舞台で力を試そう！

中学生を対象に“全国規模”でこれほど大きな模試は多くありません。
学校や県だけでなく、全国のライバルと直接比較できる貴重な機会。
『全国統一中学生テスト』は年2回、全国で一斉に無料実施されます。

全国統一中学生テストに申し込む フォームはこの下にあります

🌟 3つの特徴

Point 1｜基礎・標準・難関の3レベルで精密診断

  • 基礎：教科書〜公立高校入試の基本問題
  • 標準：公立高校入試の標準問題
  • 難関：大学入試共通テストを意識した思考力問題

👉 得意・不得意がレベル別に明確にわかります。

Point 2｜最短中5日で成績帳票返却

  • 全国／県／校舎順位
  • 偏差値・得点
  • 単元別理解度

👉 「できた／課題」がすぐに判明。解説授業と面談で復習→学習計画へ。

※「最短中5日」はデータ返却の場合です。紙の帳票返却については、校舎にお問い合わせください。

Point 3｜全国規模での比較

  • 普段は学校・地域内比較にとどまりがち
  • 全国規模での挑戦は数少ない機会

👉 岡山の会場で「全国レベル」を体験できます。

よくある質問（FAQ）

Q.費用はかかりますか？

無料です。成績帳票の返却・面談・解説授業まで含まれます。

Q.当日都合が悪い場合は？

参考受験（後日受験）をご案内します。部活動や学校行事がある方も安心してお申し込みください。

Q.持ち物は？
  • 筆記用具
  • 腕時計
  • 受験票
  • 上履き（校舎にスリッパをご用意しています。スリッパをご利用の場合は不要です）
Q.中1でも「全学年統一部門」を受けられますか？

可能です。中学全範囲が対象で、先取り学習中の方におすすめです。数学特待を希望する場合もこの部門を選択してください。

Q.点数に自信がなくても受ける意味はありますか？

全国統一中学生テストは、基礎・標準・難関の3レベルで学力を測定し、科目別に「どこまでできるか／どこが課題か」を判定。結果から“学力を伸ばす”次の一手が見えてきます。

Q.どんな中学生が受けていますか？

公立・中高一貫を含む全国の中学生です。全国比較ができる数少ない機会です。
